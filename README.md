AI-Based Employee Attendance and Salary Management System

Overview

The AI-Based Employee Attendance and Salary Management System is a web application designed to automate employee attendance using AI-powered face recognition. The system records employee attendance, maintains attendance history, and calculates salaries based on attendance records, reducing manual work and improving accuracy.

Features

-Employee Registration

-Face Registration

-Face Recognition Attendance

-Dashboard

-Attendance Reports

-Salary Management

Project Status

Frontend design has been completed, including all major pages such as Login, Dashboard, Employees, Attendance, Reports, History, and Salary. Backend development and AI integration are currently in progress.

Future Enhancements

Real-time CCTV-based attendance
Leave management
Email notifications
Payroll report generation
Cloud database integration
Tech Stack

Python, Flask, OpenCV, face_recognition, MySQL, HTML, CSS, JavaScript

pip install -r requirements.txt python app.py
